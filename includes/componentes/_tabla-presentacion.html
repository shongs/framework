1. Atributos:
Las tablas de resumen son objetos que presentan datos de contexto o de resumen de una consulta permitiendo que el usuario comprenda de mejor modo las tablas de datos que vienen a continuación o que permiten presentar las condiciones de los datos consultados. La tabla se adapta de forma líquida al contenedor.
2. Usabilidad:
Al presentar datos consolidados y no completamente ordenados se debe destacar un elemento sobre otros con color enfatizado.
3. Reglas y restricciones:
Se debe usar cuando es necesario presentar datos diversos para generar posición,contexto fechas de la consulta, etc. para otros conjuntos de datos de forma . Es recomendable ubicarla en la parte superior de la sección de contenidos antes de las tablas de datos.

Vista Mobile

En vista mobile los elementos dejan de estar flotando o posesionados generando columnas y actúan agrupados en una sola columna según los datos presentados.
